# Conversational AI in VR (Unity + ConvAI)

A **Conversational AI system integrated into Virtual Reality** using **Unity Engine** and **ConvAI**, enabling natural, real-time voice interactions with AI characters inside an immersive VR environment.

This project demonstrates how AI-driven dialogue can enhance **VR experiences** such as virtual tours, training simulations, and interactive storytelling.
